# 0. 前提ツールをインストールする

VRChat 制作で必要になるツール一式をインストールします。

## VRChat Creator Companion / ALCOM をインストールする

<Tabs>
    <TabItem value="vcc" label="VCC" default>
        [VRChat Creator Companion](https://vrchat.com/home/download/) をダウンロードし、インストールします。
    </TabItem>
    <TabItem value="alcom" label="ALCOM">
        [ALCOM](https://vrc-get.anatawa12.com/ja/alcom/) をダウンロードし、インストールします。
    </TabItem>
</Tabs>
