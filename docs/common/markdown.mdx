# マークダウンの書き方
## はじめに

これは

- Markdown ってちょっと聞いたことあるけど何? って人
- 文章を書いてみたけれど、ごちゃごちゃしていて読みづらいなぁと悩んでいる人
- 文章を簡単に構造化したい人

向けに書いた記事です!

## Markdown って?

ちょっとした記号を決まった形で置くだけで、**こんな感じに強調出来たり**

- こんな感じの
- 箇条書きができたり

1. こんな
 1. 感じの
2. 数字リスト構造を作れたり

### こんな感じでセクションを別けることが出来たり

| 名前      | 説明        |
| -------- | ----------- |
| FooBar   | Lorem Ipsum |
| HogeHuga | Lorelore    |

こんな感じのテーブルを作成できたりする、便利な文章の書き方です。

## どうやって書くの?

表示に関してはサポートされているアプリ等を利用する必要がありますが、
書くだけであれば、任意のテキストエディタで特定の記法をするだけで書くことが出来ます!

試しに、以下の構文を参考にテキストエディタで文章を書いてみてください。

```md
# マークダウン記法ガイド (一部抜粋)
## ヘッダー

# ハッシュの後にスペースを置き、文を書くとヘッダーになります
## このような形で、2段目の深いヘッダーや
### 3段目のヘッダー等も表現することができます

## リスト

### Unordered

- ハイフンの後にスペースを置き、文を書くとリストに出来ます
- このようにつなげることも出来ますが
  - スペースを2つ頭に付けると子の表現ができます
* ちなみにアスタリスクでも可です

### Ordered

1. 数字の後に.を付けてスペースを置き、文を書くと数字付きリストに出来ます。
2. リストと同じように
  1. 子の表現も出来ます

## リンク

このような形で [角括弧に囲んだ後に、括弧でリンク先を貼る](https://example.com/) とリンクを貼ったテキストを表現することができます。

## 引用

> 大なり記号を文の頭に付けてスペースを置き、文章を書くと引用の形にできます。
> > 引用をネストすることもできます。

```

これを、

- [Markdown Live Preview](https://markdownlivepreview.com/)
- [commonmark.js dingus](https://spec.commonmark.org/dingus/)
- Discord のチャット
- Teams のファイル共有 (.md ファイル)

などに貼ると、正常にフォーマットされた状態で見ることができると思います。

## それを使って書いたらどうなるの?

文章の見通しが良くなり、可読性が非常に向上します。

特殊文字等(・○①等)を用いて頑張る必要がなくなるので、変換の手間も省けます。
